# SHOW LOAD DATA

`SHOW LOAD DATA` 语句用于显示系统中创建的 LOAD DATA JOB

## 语法图

```ebnf+diagram
ShowLoadDataJobsStmt ::=
    'SHOW' 'LAOD' 'DATA' 'JOBS'

ShowLoadDataJobStmt ::=
    'SHOW' 'LAOD' 'DATA' 'JOB' JobID
```

SHOW LOAD DATA JOBS;
